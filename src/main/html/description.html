Add native system notifications to the IDE

What does this plugin do?

It turns important IDE events into native operating system notifications. This way, you can follow the status of your tasks without keeping the IDE in the foreground.


Key features:
  • Shows external system tasks (Gradle, Maven, ...) as system notifications when they finish.
  • Notifies when Run/Debug configurations start and finish.
  • Optional: mirror all IDE notifications as system notifications.
  • Option to add sound to notifications.
  • OS compatibility:
    • Windows: Action Center (tray).
    • macOS: Notification Center.
    • Linux: libnotify; with an in-IDE fallback when there is no D-Bus session.
  • Easy configuration under Settings/Preferences → Tools → System Notifier.

Notes
  • It does not collect or send personal data.
  • On Linux a D-Bus session is required for native notifications; otherwise, an in-IDE balloon is used.
