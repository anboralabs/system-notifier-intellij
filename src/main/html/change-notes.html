Versions:
  • All IntelliJ products support: 1.1.0

Release notes:
  • 1.1.0 (2025-10-26) — error reporting and maintenance
    • New: "Report to Anbora-Labs" error reporter integrated (Sentry-based).
    • Improved: Error details are sent in the background; server/host name is omitted for privacy.
    • Chore: Update plugin metadata and dependencies; bump version to 1.1.0.
  • 1.0.2 (2025-09-24) — upgrade sdk
  • 1.0.1 (2025-09-09) — change notification title
  • 1.0.0 (2025-07-31) — initial release
    • System notifications for Gradle and Maven tasks when they finish.
    • Notifications when executions (Run/Debug) start and finish.
    • Option to mirror all IDE notifications as system notifications.
    • Option to play a sound with notifications.
    • Compatibility: Windows (Action Center), macOS (Notification Center), Linux (libnotify) with an in-IDE fallback if there is no D-Bus session.
    • Settings panel under Settings/Preferences → Tools → System Notifier.
